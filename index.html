Efficient FrontierThe set of portfolios that offer the highest expected return for each level of risk. Based on Markowitz mean-variance optimization (1952).

1,000 random weight combinations are generated and plotted in risk-return spaceX-axis = annualized volatility (risk). Y-axis = expected return. Each dot is one possible portfolio.. The upper-left edge of the cloud is the efficient frontierPortfolios that maximize return for a given level of risk — or minimize risk for a given return. Nothing above or to the left is achievable..

Optimal Portfolio WeightsThe portfolio with the highest Sharpe ratio — best risk-adjusted return. Found where the Capital Market Line is tangent to the frontier.
